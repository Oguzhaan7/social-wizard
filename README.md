# Social Wizard

## Technologies Used

- **Fastify**: A fast and low-overhead web framework for Node.js.
- **RabbitMQ**: A robust messaging broker for communication between services.
- **WebSocket**: A protocol for real-time communication between the server and the client.
- **MongoDB**: A NoSQL database for storing and retrieving application data.
- **Nuxt.js**: A framework for building server-side rendered and static Vue.js applications.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs quickly.
